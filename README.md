# プロフィールサイト模写

## 概要
既存のプロフィールサイトを参考に、HTML/CSS/JavaScriptを用いて模写制作しました。
レスポンシブ対応を行い、スマホでも閲覧し易いよう調整しています。

## 制作目的
- Figmaの操作練習の為
- HTML/CSSでのレイアウト構築に慣れる為
- JavaScript/jQueryを使った動きを実装する為
- 実務でありそうなレスポンシブ対応を意識する為

## 使用技術
- HTML
- CSS
- JavaScript(basic)
- jQuery
- Figma
- Canva

## 担当範囲
- デザインカンプを元にコーディング
- レスポンシブ対応
- スムーススクロール
- ハンバーガーメニュー

## 工夫した点
- 技術と知識向上の為、画像加工(丸型加工)はCSSで行いました
- 全体の配色が合うように、ヒーロー画像(メインビジュアル)はCanvaで製作
- レスポンシブデザインではハンバーガーメニューを導入しました(追加)

## 苦労した点・解決方法
- レスポンシブデザイン作成に不慣れな為、バランス調整が大変でした
- わからない部分は書籍やネットで検索して解決しました
- それでもわからない場合は、AI(Copilot)を使用して解決しました

## 今後改善したい点
- お借りしたコードを自分で書けるように勉強
- 無駄にCSSを指定してしまっていて、流用(再利用)出来る部分は再利用したいです(余白だけ追加指定にするなど)

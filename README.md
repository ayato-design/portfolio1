# プロフィールサイト模写

## 概要
デザイン・コーディング練習として制作した、プロフィールサイトの模写作品です。
Figmaでの構成設計からHTML / CSSによるコーディングまでを行いました。

## 制作目的
HTML / CSS の基礎理解と、レイアウト・余白設計の練習を目的に制作しました。
その他、Figma操作練習も兼ねています。

## 工夫した点
- 余白とタイポグラフィを意識
- シンプルで見やすい配色
- レスポンシブ対応（スマートフォン表示）
- CSSで画像を丸型加工
- 画像をCanvaで作成・加工

## 使用技術
- HTML
- CSS
- JavaScript(basic)
- jQuery
- Figma
- Canva
